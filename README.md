
# GO-bookstore

A simple GO REST API implementation

- golang
- REST
- mssql
- CRUD operations





## Execute

To run this project

- Make a .env file with the database info


```bash
  touch .env

  SQL_CONNECTION_STRING=<your_DB_string_here>

```

```bash
  cd cmd/main
  go run main.go
```
